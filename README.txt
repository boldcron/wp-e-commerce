Boldcron WP e-commerce
Versão 1.2.0

O módulo de integração BoldCron para wp-ecommerce, implementa na sua loja virtual o meio de pagamento da BoldCron de forma simples. Veja como é facil a implementação deste plugin.

Baixe a biblioteca
wp-ecommerce.zip

Pré-requisitos
Wordpress com o plugin WP e-commerce
Bibliotecas PHP
SOAP
DOMDocument
SimpleXML 

Você pode fazer a verificação dessas bibliotecas, usando o método phpinfo() no seu código PHP.
Configuração
1° Descompacte os arquivos da biblioteca dentro da pasta /wp-content/plugins/wp-e-commerce/merchants/, se esta for a primeira vez que estiver instalando o módulo, nenhum arquivo será sobrescrito, caso seja uma atualização, clique em Mesclar/Substituir tudo.
2° Acesse o painel administrativo da sua loja wp-ecommerce, localize “Store” no menu à esquerda e clique em “Settings”.
3° Em seguida, clique no menu “Payment Options”.
4° Na tela que for exibida, marque a opção de pagamento “BoldCron”
5° E para finalizar, definas a configurações da sua loja BoldCron, e clique no botão “Update”
